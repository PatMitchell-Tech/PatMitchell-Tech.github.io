I have been doing a home lab and self hosting for almost 2 years. I self host and use a homelab as a means to increase my knowledge of corporate environments and to run certain services in my network.

How my homelab is setup:

Infrastucture and technical details of my homelab:

Raspberry pi 4 model B:
Broadcom BCM2711, Quad core Cortex-A72 (ARM v8) 64-bit SoC @ 1.8GHz
8 Gb ram

Asrock desktop:
Nvidia RTX geforce 3060 ti graphics card
2x 16gb ddr4 sticks of ram
AMD Ryzen 5 5600X 6-Core Processor, 3.7 Mhz, 6 cores, 12 logical processors

Services currently being ran:
My raspberry pi 4 is currently running SSH for SFTP, and pi-hole and a web admin server to control the server. I also am running a docker container on my raspberry pi to host my homepage to manage all my services.
My desktop is currently running two VMs one for a windows server 2022 domain controller and one windows 10 pro connnected client
